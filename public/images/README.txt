Place your portfolio images here, using these exact filenames (or update the
paths in src/data/content.ts to match whatever you use):

- profile-photo.png        (your headshot, used in the hero profile card)
- simpu-main.png           (SIMPU project main screenshot)
- simpu-secondary.png      (SIMPU project secondary screenshot)
- taters-main.jpg          (Taters project screenshot)

If an image fails to load, the UI gracefully falls back to a placeholder
block (for project shots) or your initials (for the avatar) — see the
onError handlers in Hero.tsx and Projects.tsx.
